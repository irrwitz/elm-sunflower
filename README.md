# Sunflower
A sunflower implementaton in [elm](http://elm-lang.org).

## Run locally
 * Checkout via `git clone https://github.com/irrwitz/elm-sunflower`
 * Run `elm-reactor` in the directory `elm-sunflower`
 * Open browser at `localhost:8000`
